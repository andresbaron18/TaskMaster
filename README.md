## TaskMaster
TaskMaster is a project management web application built from scratch using vanilla JavaScript, HTML, and CSS.

This project was developed as part of my Full Stack Developer learning process to practice DOM manipulation, LocalStorage, and dynamic user interfaces without using frameworks.

## Features
- Create projects
- Edit projects
- Delete projects
- Dynamic UI rendering
- Modal forms
- LocalStorage data persistence
- Responsive layout
- Custom UI design
  
## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- Git
- GitHub
